# 📓 Notepad App

Bloco de notas minimalista feito com React + Vite.

## Funcionalidades

- ✅ Criar, editar e apagar notas
- ✅ Busca em tempo real
- ✅ Salva automaticamente no `localStorage`
- ✅ Contador de palavras e caracteres
- ✅ Cores diferentes por nota
- ✅ Visual clean e responsivo

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173

## Build para produção

```bash
npm run build
```
